SMILE GAME BUILDER で制作されたゲームの操作方法

-----------------------------------------------------------------------
キーボード操作
-----------------------------------------------------------------------
方向キー / [A][D][W][S]キー
移動

[Shift]キー + 移動
ダッシュ移動 (主人公視点のときはカメラを自動回転しない)

[ESCキー] / [X]キー
メニュー表示 / キャンセル

[Enter]キー / [Z]キー
決定

[E][Q][R][F]キー
カメラ回転

[C][V]キー
カメラズーム

[B]キー
カメラ切り替え(標準視点 / 一人称視点)

[K]キー
カメラ位置リセット

[F4]キー
ウィンドウサイズ切り替え



-----------------------------------------------------------------------
ゲームパッド操作（Xbox 360 コントローラ対応）
-----------------------------------------------------------------------
[左スティック] / 方向ボタン
移動

[Xボタン] + 移動
ダッシュ移動 (主人公視点のときはカメラを自動回転しない)

[Aボタン]
メニュー表示 / キャンセル

[Bボタン]
決定

[右スティック]
カメラ回転

[LBボタン] / [RBボタン]
カメラズーム

[Yボタン]
カメラ切り替え(標準視点 / 一人称視点)

[右スティック]押し込み
カメラ位置リセット
